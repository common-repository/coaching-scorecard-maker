== Description ==

Coaching Scorecard Maker is a simple tool that helps you create interactive scorecards, quizzes and assessments for your audience. 

In addition to computing the total score, you can provide a detailed, customized analysis of the results to showcase your expertise and guide your audience to different offerings.

Features include:

* Simple shortcode based approach to create scorecards on blog posts and pages
* Full control over analysis page, including displaying total score, group scores or individual item scores
* Provide customized analysis based on user input

== Installation ==

To install the plugin manually:

1. Extract the contents of the archive (zip file)
2. Upload the entire coaching-scorecard-maker folder to your '/wp-content/plugins' folder
3. Activate the plugin through the Plugins section in your WordPress admin
